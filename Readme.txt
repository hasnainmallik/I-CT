This project is a personal portfolio website built using HTML, CSS, and JavaScript. It showcases the skills, projects,
 and contact information of a web developer named Hasnain Malik. The website features a responsive design with a 
navigation bar, sections for "About Me," "Skills," and "Projects," and a contact form with validation. 
JavaScript is used to display a welcome alert, change the background color dynamically, and validate the contact form. 
The CSS ensures a clean and modern layout, with a fixed footer and smooth transitions for interactive elements. 
Overall, it serves as a professional and interactive showcase of the developer's work and capabilities.
